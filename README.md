# Netflix Content Analytics Dashboard

**Stack:** Power BI · DAX · Power Query

## Overview
An interactive Power BI dashboard analyzing Netflix's content catalog — modeling relationships across titles, genres, ratings, release years, and countries of origin.

## What it does
- Cleans and transforms raw catalog data in Power Query for accurate, drill-down analysis
- Builds a data model connecting titles to genres, ratings, release years, and countries
- Surfaces content trends, genre distribution, and regional catalog composition through DAX measures
- Presents findings in an executive-style reporting layout

## Key skills demonstrated
- Power Query data cleaning and transformation
- Data modeling (relationships across catalog dimensions)
- DAX measure development
- Dashboard design for executive/non-technical audiences
